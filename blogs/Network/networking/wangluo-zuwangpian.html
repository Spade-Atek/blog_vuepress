家庭组网方案

交流学习参考恩山无线论坛 https://www.right.com.cn/forum/open in new window

1. 单个路由器

技术架构

  • 核心设备：一台无线路由器，通常支持Wi-Fi 5（802.11ac）或Wi-Fi 6（802.11ax）。
  • 信号覆盖：依赖路由器本身的Wi-Fi天线发射无线信号，覆盖范围一般为 50-100㎡（实际受钢筋混凝土的影响会更少），穿墙能力有限。
  • 有线连接：
    • 光猫（ONU）→ 路由器WAN口
    • 路由器LAN口可连接PC、交换机、智能家居设备

关键技术

  • Wi-Fi 标准：
    • Wi-Fi 5（802.11ac）：5GHz频段，速度可达1300Mbps，但穿墙能力较差。
    • Wi-Fi 6（802.11ax）：更高带宽（2400Mbps+），支持OFDMA、BSS Coloring，提高多设备连接性能。
  • MU-MIMO（多用户多入多出）：允许路由器同时与多个设备通信，提高效率。
  • 波束成形（Beamforming）：增强信号朝向，提高覆盖范围和稳定性。

优缺点

✅ 优点：

  • 价格便宜，安装简单
  • 适合小户型或单房间使用 ❌ 缺点：
  • 信号覆盖范围有限，穿墙后衰减明显
  • 多设备连接后性能下降

2. AC+AP（有线AP）方案

适用于大户型、别墅、办公环境，提供专业级Wi-Fi覆盖。

技术架构

  • 核心设备：
    • AC（无线控制器）：统一管理多个AP（Access Point）。
    • AP（无线接入点）：提供Wi-Fi信号，可选吸顶式、面板式、桌面式等。
    • PoE交换机：为AP供电并传输数据，避免额外布电源线。
    • 有线回程：所有AP通过网线连接到PoE交换机。
  • 信号覆盖：
    • 通过多个AP合理布局，实现全屋覆盖
    • AC控制器统一管理，实现无缝漫游

关键技术

  • PoE（以太网供电）：AP只需一根网线即可同时供电和传输数据，减少布线复杂度。
  • 无缝漫游：
    • 802.11r（Fast Roaming）：加快设备切换AP的认证过程，提高漫游速度。
    • 802.11k（Radio Resource Management）：让设备提前知道最优AP，减少不必要的切换。
    • 802.11v（BSS Transition）：引导设备选择最合适的AP。
  • Wi-Fi 6/6E 及 AP 自动信道优化：降低干扰，提高整体网络质量。

3. Mesh 路由器方案

适用于无网线布线条件的中大户型用户（个人认为有线回程也ok，适合有网线布线条件），采用无线回程或有线回程方式连接多个路由器。

技术架构

  • 核心设备：Mesh 主路由 + 多个Mesh节点
  • 回程方式：
    • 无线回程：Mesh节点之间通过5GHz频段通信（如Wi-Fi 6 2402Mbps链路）。
    • 有线回程（推荐）：Mesh节点之间用网线连接，减少无线干扰，提高性能。
  • 设备连接：
    • 光猫 → 主路由器 → 子路由器（无线/有线） → 终端设备

关键技术

  • 动态路由协议（802.11s）：Mesh设备间动态选择最优路径，保证稳定性。
  • 智能漫游：
    • 类似AC+AP方案，但由Mesh系统自动管理，无需独立AC
    • 支持 802.11k/v/r 协议，提高切换效率。
  • Wi-Fi 6/6E 支持：
    • 采用 OFDMA 和 MU-MIMO 让多设备高效通信。
    • 160MHz 频宽，提高速率。

4. 电力猫（PLC）+AP方案

适用于老房、无网线布线环境，利用电力线传输网络数据。

技术架构

  • 核心设备：
    • 主电力猫（PLC）：连接光猫或主路由器，发送数据到电力线路。
    • 子电力猫（PLC）：插入其他房间插座，转换成有线或无线信号。
    • AP（可选）：扩展无线覆盖范围。

关键技术

  • HomePlug AV2 标准：支持千兆传输，低延迟。
  • MIMO over Powerline：提高信号稳定性，减少干扰。
  • 128-bit AES 加密：保障数据安全。

优缺点

✅ 优点：

  • 免布线，适合装修完毕的家庭
  • 适用于有多个插座的房间 ❌ 缺点：
  • 受电力环境影响较大，旧房电路可能干扰严重
  • 速度比直连网线方案低

5. FTTR（光纤到房间）

适用于别墅、大户型，提供最高速率的家庭组网方案。

技术架构

  • 核心设备：
    • 光猫（OLT）
    • 主FTTR光纤网关
    • FTTR子网关（AP功能）
  • 传输方式：
    • 光纤到房间（FTTR），主光猫通过光纤连接各房间子网关，实现千兆/万兆连接。
    • Wi-Fi 6/7全屋覆盖，所有子网关提供稳定无线网络。

关键技术

  • 10G PON 光纤技术：提供千兆/万兆速率。
  • Wi-Fi 7（未来）：
    • 320MHz 频宽 提供更高速率
    • Multi-Link Operation（MLO） 提高稳定性

优缺点

✅ 优点：

  • 网络极致稳定，适合大带宽需求用户
  • 未来可升级至10Gbps级别 ❌ 缺点：
  • 施工复杂，成本昂贵
  • B站有UP主司波图有布过视频，只能说适合装修之前布设，不适合明走，如果明走可以看看隐形光纤

总结

方案适用场景主要优势主要缺点关键技术
单路由器小户型成本低，安装简单覆盖有限Wi-Fi 6, MU-MIMO
AC+AP大户型, 新装修专业级漫游, 稳定需布线PoE, 802.11k/v/r
Mesh大户型, 无网线免布线, 组网灵活无线回程受干扰802.11s, OFDMA
PLC+AP老房, 免布线无需布网线受电力环境影响HomePlug AV2
FTTR豪宅, 极致体验千兆/万兆网络施工复杂，贵10G PON, Wi-Fi 7

软路由 vs 硬路由

硬路由（传统路由器）

  • 定义：就是我们常见的家用路由器，硬件固定，功能简单。
  • 优点：
    • 简单易用，插电就能用
    • 稳定性高，适合日常家庭上网
    • 节能省电
  • 缺点：
    • 功能比较有限，不能做很多高级设置
    • 性能不如高端设备，处理大流量时会有瓶颈
  • 适用场景：家庭上网、普通办公
  • 例子：TP-Link、华硕、小米等家用路由器

✅ 软路由（自建路由）

  • 定义：使用一台小电脑（如 N5105 小主机）加上特定操作系统（如 OpenWRT 或 pfSense）来做路由器。
  • 优点：
    • 性能强，可以处理更大流量
    • 功能多，支持高级设置，比如防火墙、VPN、负载均衡等
    • 灵活可定制，可以根据需求做很多调整
  • 缺点：
    • 配置较复杂，需要一定的技术基础
    • 需要额外购买硬件（主机、无线设备等）
    • 比硬路由消耗更多电力
  • 适用场景：需要高性能、更多自定义设置的用户（如高端家庭、企业）
  • 例子：小型计算机刷开源固件如OpenWRT、爱快
  • 刷过爱快的软路由
